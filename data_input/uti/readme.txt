Steps to download utility data:

Download book (open access):
Szende A, Janssen B, Cabases J, eds. Self-Reported Population Health: An International Perspective Based on EQ-5D. Springer Netherlands; 2014. doi:10.1007/978-94-007-7596-1

Place book in this folder

Run the script pdf_extract.R
Select the tables in the PDF viewer that pops up
When prompted, input the type of value set, or type NONE to skip the current tables

An RDS file will be placed in this folder
